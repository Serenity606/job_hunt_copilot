# Job Hunt Co-Pilot

Track job applications, analyze responses

## What It Does

- Log applications (company, role, status, date)
- See stats (applied, interviews, offers, rejections)
- Daily motivation quotes
- Works entirely in browser (local storage)

## Why I Built This

A simple way to track my progress without spreadsheets.

## Tech Stack

- Python + Flask
- HTML/CSS/JavaScript
- JSON file storage
